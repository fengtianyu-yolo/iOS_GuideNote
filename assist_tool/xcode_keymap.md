# XCode的使用

## 常用的快捷键

* 移动焦点到其他区域 

    ```
    option + command + `
    ```

* 跳转到文件筛选框位置

    ```
    option + command + J
    ```

* 光标移动到编辑区

	```
	ctrl + `
	```

* 光标移动到日志输出控制台区域 

	```
	command + shift + c
	```

* 多点编辑 

    ```
    ctrl + shift + 点击的位置
    ```
    
* 光标移动到上一次编辑的位置

	```
	ctrl + |
	```

* 光标移动到下一次编辑的位置

	```
	ctrl + shift + |
	```
